## データ可視化の重要性
データ可視化は、複雑なデータを直感的に理解するために重要なプロセスです。文字や数値だけで表現されたデータは、そのままでは意味を捉えにくいことがありますが、グラフやチャートを用いて適切に可視化することにより、データの傾向やパターンを迅速に把握することが可能となります。さらには、高精度な分析や、意思決定の効果的なサポートにも繋がります。

### Python によるデータ可視化
Python には、データ可視化を行うためなツールがさまざま提供されています。特に、オープンソースのライブラリである `Matplotlib` や `Seaborn` は、柔軟で詳細なカスタマイズが可能なグラフ作成を強力にサポートしてくれます。これらのライブラリを使用することで、簡単な散布図から複雑なヒートマップや3Dプロットまで、多種多様な可視化を容易に実現できます。

今回学ぶ `Matplotlib` は、Python で最も広く使われている定番の可視化ライブラリです。細かいカスタマイズが自由自在にできる反面、最初は少し面倒に感じるところもあるかもしれませんが、Python によるデータ可視化を学ぶうえで欠かせません。

公式：<a href="https://matplotlib.org/" target="_blank">https://matplotlib.org/</a>

<div class="information-box common-icon-box">通常、このような可視化ライブラリは <code>pip</code> コマンドでインストールする必要がありますが、<strong>CHIKUWA Editor には予めインストール済み</strong>ですので、特別な操作は不要です。外部の開発環境を利用する場合は、必要に応じて各自でインストールしてください。</div>
<h2>Matplotlib の基本</h2>
<h3>Matplotlib のインポート</h3>
<code>Matplotlib</code> による可視化は、<code>pyplot</code> というサブモジュールを通して行われます。インポートは以下のようにして行います。ソースコードの冒頭で一度だけ書くようにしましょう。
<pre class="wp-block-code python"><code>import matplotlib.pyplot as plt</code></pre>
以下で紹介するコードは、<code>matplotlib.pyplot</code> モジュールがインポートされ、<code>plt</code> という省略表記で利用できる状態であることを前提としています。
<h2>基本的なグラフの描画</h2>
<h3>折れ線グラフ</h3>
データをプロットして折れ線グラフを表示するには、以下のように <a href="https://matplotlib.org/stable/api/_as_gen/matplotlib.pyplot.plot.html#matplotlib.pyplot.plot" target="_blank"><code>plot()</code></a> 関数と <a href="https://matplotlib.org/stable/api/_as_gen/matplotlib.pyplot.show.html" target="_blank"><code>show()</code></a> 関数を用います。
<pre class="wp-block-code python"><code># x の値（横軸）
x = [0, 1, 2, 3, 4]
# y の値（縦軸）
y = [0, 1, 4, 9, 16]
# データをプロット
plt.plot(x, y)
# グラフの表示
plt.show()</code></pre>

以下のように1行に収めることもできます。
<pre class="wp-block-code python"><code>plt.plot([1, 2, 3], [1, 4, 9])</code></pre>

### ヒストグラム
ヒストグラムを表示するには、[`hist()`](https://matplotlib.org/stable/api/_as_gen/matplotlib.pyplot.hist.html)  関数を用います。ヒストグラムではデータの他に、区間（ビン）の数（`bins`）やビンの色（`color`, `edgecolor`）を指定することができます。

```py
import numpy as np
import matplotlib.pyplot as plt

# 標準正規分布のデータを 300 個生成
data = np.random.randn(300)

# ヒストグラムを描画
plt.hist(
    data, 
    bins=20, 
    color='skyblue', 
    edgecolor='black'
)
plt.xlabel("Value")
plt.ylabel("Frequency")
plt.show()
```

<h3>グラフの基本要素</h3>
グラフの基本要素として、次のものがあります。
<ul>
 	<li>「タイトル 」(<code>title</code>)</li>
 	<li>「横軸ラベル」(<code>xlabel</code>)</li>
 	<li>「縦軸ラベル」(<code>ylabel</code>)</li>
 	<li>「グリッド線」(<code>grid</code>)</li>
 	<li>「凡例」(<code>legend</code>)</li>
</ul>
それぞれ以下のようにして使うことができます。
<pre class="wp-block-code python"><code>plt.plot(
    [1, 2, 3], [1, 4, 9], 
    label="Sugoi Sen"          # 凡例のラベル
)
plt.title("Kakkoii Graph") # グラフタイトル
plt.xlabel("Yoko Label")   # 横軸ラベル
plt.ylabel("Tate Label")   # 縦軸ラベル
plt.grid(True)             # グリッド線の表示
plt.legend()               # 凡例の表示</code></pre>

<h3>日本語フォントの利用</h3>
グラフのタイトルや軸ラベルに日本語を使いたい場合は、<code>rcParams["font.family"]</code> で日本語フォントを指定します。一般的な Python 環境では日本語フォントはインストールされていませんが、CHIKUWA Editor では <code>Noto Sans CJK JP</code> をインストール済みですので、以下のようにして使うことができます。
<pre class="wp-block-code python"><code>plt.rcParams["font.family"] = "Noto Sans CJK JP"</code></pre>
日本語フォントを指定した状態であれば、以下のようにグラフのタイトルや軸ラベルに日本語を使うことができます。

<h2>NumPy の関数</h2>
データ可視化の練習をするうえで知っておくと便利な <code>NumPy</code> の関数をいくつか紹介します。必要に応じて <a href="https://numpy.org/ja/" target="_blank">NumPy 公式</a>のドキュメントを参照するようにしてください。
<h3>データ生成系</h3>
<ul>
 	<li><a href="https://numpy.org/doc/stable/reference/generated/numpy.linspace.html#numpy-linspace" target="_blank"><code>np.linspace(start, stop, num)</code></a>：指定範囲内で等間隔な値の配列を生成</li>
 	<li><a href="https://numpy.org/doc/stable/reference/generated/numpy.arange.html" target="_blank"><code>np.arange(start, stop, step)</code></a>：指定範囲内でステップ幅を持つ配列を生成</li>
 	<li><a href="https://numpy.org/doc/stable/reference/random/generated/numpy.random.rand.html#numpy-random-rand" target="_blank"><code>np.random.rand()</code></a>：一様分布の乱数を生成</li>
 	<li><a href="https://numpy.org/doc/stable/reference/random/generated/numpy.random.randn.html#numpy-random-randn" target="_blank"><code>np.random.randn()</code></a>：標準正規分布の乱数を生成</li>
 	<li><a href="https://numpy.org/doc/stable/reference/random/generated/numpy.random.normal.html#numpy-random-normal" target="_blank"><code>np.random.normal()</code></a>：正規分布の乱数を生成</li>
 	<li><a href="https://numpy.org/doc/stable/reference/random/generated/numpy.random.seed.html#numpy-random-seed" target="_blank"><code>np.random.seed()</code></a>：乱数のシード値を設定（再現性のある乱数を生成したい場合）</li>
 	<li><a href="https://numpy.org/doc/stable/reference/generated/numpy.zeros.html#numpy.zeros" target="_blank"><code>np.zeros()</code></a>：0 で満たされた配列を作成</li>
 	<li><a href="https://numpy.org/doc/stable/reference/generated/numpy.ones.html#numpy.ones" target="_blank"><code>np.ones()</code></a>：1 で満たされた配列を作成</li>
</ul>
<h3>数学・統計系</h3>
<ul>
 	<li><code>np.sin()</code>, <code>np.cos()</code>, <code>np.exp()</code>：さまざまな数学関数</li>
 	<li><code>np.mean()</code>, <code>np.median()</code>, <code>np.std()</code>：基本統計量の計算</li>
 	<li><code>np.min()</code>, <code>np.max()</code>：最小値・最大値の取得</li>
 	<li><code>np.pi</code>：円周率 π<code></code></li>
</ul>
